HOW IT WORKS
================================================================================

USER CONNECTS INSTAGRAM
        ↓
autoSync SCRAPES VIA APIFY
        ↓
PROFILE DATA SAVED TO FIRESTORE (settings/profile)
        ↓
PAGE LOADS, CALLS useInsights()
        ↓
HOOK LOADS PROFILE FROM FIRESTORE
        ↓
InsightsEngine GENERATES INSIGHTS FROM REAL DATA
        ↓
COMPONENT RENDERS PERSONALIZED INSIGHTS
        ↓
NO HARDCODED RECOMMENDATIONS ANYWHERE ✓
