USER WORKFLOW
================================================================================

1. User navigates to "Competitor Intel" from Sidebar
2. Page loads, their profile auto-populates from Firestore
3. User enters competitor Instagram username (with or without @)
4. User clicks "Analyze" button
5. ApifyService scrapes competitor profile (loading state shown)
6. Competitor data displayed in comparison grid
7. Action plan automatically generates (7-8 insights)
8. User can add more competitors (repeating steps 3-7)
9. User reviews color-coded metrics and priorities
10. User can remove competitors (updates plan automatically)
11. Competitor data persists in localStorage across sessions

Error Scenarios Handled:
  - No Apify token: Alert user to configure in Settings
  - Username not found: Show friendly error, allow retry
  - API failure: Catch error, retry option
  - Missing Firestore profile: Show empty state gracefully
  - Malformed localStorage: Fallback to empty competitors array
